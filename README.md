# 電子レンジ 温め時間変換器

食品パッケージに記載されている電子レンジの温め時間を、お使いの電子レンジのワット数に合わせて変換するツールです。

## 機能

- パッケージ記載のワット数と時間から、お使いの電子レンジでの適切な温め時間を計算
- よく使用されるワット数（500W, 600W, 700W, 800W）をワンタッチで入力可能
- 分秒での時間入力と表示に対応

## 計算方法

以下の計算式で温め時間を算出します：

```
実際の温め時間 = 記載されたワット数 ÷ お使いの電子レンジのワット数 × 記載された温め時間
```
